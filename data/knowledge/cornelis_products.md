# Cornelis Networks Product Knowledge

## Overview

Cornelis Networks provides high-performance fabric solutions for HPC and AI/ML workloads.

## Product Areas

### Omni-Path Express (OPX)
- High-performance fabric for HPC clusters
- Low latency, high bandwidth interconnect
- Supports large-scale deployments

### Fabric Manager
- Management software for Omni-Path fabrics
- Topology discovery and visualization
- Performance monitoring and diagnostics

### Host Software Stack
- Drivers for Linux operating systems
- PSM2 (Performance Scaled Messaging)
- MPI integration

## Components

### Firmware
- Switch firmware
- HFI (Host Fabric Interface) firmware
- Management controller firmware

### Drivers
- Kernel drivers (hfi1)
- User-space drivers
- RDMA/Verbs support

### Tools
- opainfo - Fabric information utility
- opareport - Reporting and diagnostics
- opafm - Fabric Manager CLI

### Documentation
- Installation guides
- User manuals
- API documentation
